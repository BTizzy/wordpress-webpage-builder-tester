================================================================================
INTERACTION STATES
================================================================================

BUTTONS:
  Normal:    ⬛ Black background, white text
  Hover:     ⬛ Lifts up 2px, gold border appears, shadow increases
  Active:    ⬛ Pressed down, darker shade

NAVIGATION LINKS:
  Normal:    Grey text
  Hover:     Black text, gold underline appears
  Active:    Black text, gold underline stays

CARDS:
  Normal:    White background, subtle shadow
  Hover:     Lifts up 4px, shadow increases, border turns gold

FORMS:
  Input Normal:   Light grey background, medium grey border
  Input Focus:    White background, gold border
  Input Error:    Light red background, red border
  Input Success:  Light green background, green border

================================================================================
ANIMATIONS
================================================================================

PAGE LOAD:
  ✓ Hero section fades in (0.8s)
  ✓ Navigation slides down from top (0.5s)

SCROLL ANIMATIONS:
  ✓ Sections fade in and slide up when scrolled into view
  ✓ Cards have stagger effect (appear one by one, 100ms delay)
  ✓ Smooth transitions, not jarring

HOVER ANIMATIONS:
  ✓ Buttons lift up with shadow (0.3s ease)
  ✓ Cards scale up slightly (1.02x) with shadow
  ✓ Links get gold underline that slides in from left

NAVIGATION:
  ✓ Hamburger menu slides in from top on mobile
  ✓ Menu items fade in with stagger
  ✓ Smooth scrolling to anchor links

================================================================================
TYPOGRAPHY HIERARCHY
================================================================================

H1 (Hero Title):
  • Font: Inter, Bold (700)
  • Size: 56px desktop, 32px mobile
  • Color: Black
  • Line Height: 1.2

H2 (Section Titles):
  • Font: Inter, Bold (700)
  • Size: 40px desktop, 28px mobile
  • Color: Black
  • Line Height: 1.2

H3 (Card Titles):
  • Font: Inter, Semi-Bold (600)
  • Size: 28px desktop, 22px mobile
  • Color: Black
  • Line Height: 1.2

Body Text:
  • Font: Inter, Regular (400)
  • Size: 16px (all devices)
  • Color: Dark Grey
  • Line Height: 1.6

Small Text (Footer):
  • Font: Inter, Regular (400)
  • Size: 14px
  • Color: Light Grey
  • Line Height: 1.6

================================================================================
